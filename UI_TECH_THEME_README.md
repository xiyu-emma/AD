# UI 科技感主題更新說明

## 概述
將口述影像生成系統的使用者介面改造成現代化科技感深色主題，靈感來自 GitHub Dark、VS Code Dark 等現代開發工具。

## 主要改動

### 1. 配色方案（深色主題）
- **背景色系**
  - 主背景: `#0D1117` (深太空灰)
  - 卡片背景: `#161B22` (次深灰)
  - 更深背景: `#010409` (用於文本框和狀態列)

- **主題色**
  - 主要藍: `#58A6FF` (科技藍，用於標題和主要按鈕)
  - 次要藍: `#388BFD` (深藍，用於次要按鈕)
  - 強調色: `#F778BA` (霓虹粉，用於重要操作)
  - 成功綠: `#3FB950` (用於完成狀態)
  - 懸停綠: `#238636` (按鈕懸停效果)

- **文字色系**
  - 亮灰文字: `#C9D1D9` (主要文字)
  - 暗灰文字: `#8B949E` (次要文字、提示)
  - 邊框色: `#30363D` (分隔線、邊框)

### 2. 視覺增強

#### 標題區域
- 增大標題字體 (28pt → 32pt)
- 使用科技藍色標題，暗灰色副標題
- emoji 與文字之間增加空格
- 調整分隔線顏色和間距

#### 按鈕設計
- 增大按鈕內邊距和字體 (12pt → 13pt)
- 添加高亮邊框效果 (`highlightthickness=2`)
- 改進懸停效果 (使用成功綠色)
- 深色文字配淺色按鈕背景，提升對比度

#### 預覽區域
- 卡片使用深色背景，增加內邊距 (15px → 18px)
- 預覽標籤使用更深背景 (`COLOR_BG_DARKER`)
- 文本框高亮邊框，聚焦時顯示主題藍色
- 標題左對齊，增加視覺層次

#### 即時攝影機窗口
- 深色背景配科技藍標題
- 增加視窗尺寸 (640x640 → 680x720)
- 改進狀態標籤設計

#### 工具提示 (Tooltip)
- 深色背景 `#1C2128`
- 添加科技藍邊框高亮
- 增加內邊距，改善可讀性

### 3. Sun Valley 主題整合
- 從 `light` 主題切換到 `dark` 主題
- 保持 ttk 組件與自定義配色一致
- 進度條使用科技藍配色

### 4. 狀態列與音色選擇區
- 狀態列使用最深背景色
- 音色選擇區使用卡片背景色
- 改進標籤和間距設計

## 技術細節

### 顏色常量
```python
COLOR_BG_MAIN = "#0D1117"        # 深太空灰背景
COLOR_BG_CARD = "#161B22"        # 卡片背景
COLOR_BG_DARKER = "#010409"      # 更深的背景
COLOR_PRIMARY = "#58A6FF"        # 科技藍
COLOR_SECONDARY = "#388BFD"      # 深藍
COLOR_ACCENT = "#F778BA"         # 霓虹粉
COLOR_SUCCESS = "#3FB950"        # 成功綠
COLOR_WARNING = "#D29922"        # 警告黃
COLOR_TEXT_BRIGHT = "#C9D1D9"    # 亮灰文字
COLOR_TEXT_DIM = "#8B949E"       # 暗灰文字
COLOR_BORDER = "#30363D"         # 邊框顏色
COLOR_HOVER = "#238636"          # 懸停綠
```

### 主要修改文件
- `main.py`: 所有 UI 相關配色和樣式

### 相容性
- 保持所有功能不變
- 與現有語音克隆、模型預載入等功能完全相容
- 適用於 Windows、Linux、macOS

## 視覺效果對比

### 修改前（淺色溫暖主題）
- 米色背景 (#F2D9BB)
- 柔和的藍色按鈕
- 較低對比度
- 傳統辦公軟體風格

### 修改後（深色科技主題）
- 深灰/黑色背景
- 高對比度科技藍按鈕
- 霓虹色強調
- 現代開發工具風格

## 使用方式
直接運行 `python main.py` 即可看到新的科技感介面。
