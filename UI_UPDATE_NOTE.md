# 🎨 UI 主題更新通知

## 重要更新

本專案已整合 **Sun Valley ttk theme** 淺色主題，大幅提升使用者介面品質。

## 🚀 快速開始

### 1. 安裝新依賴
```bash
pip install sv-ttk
```

或使用完整依賴清單：
```bash
pip install -r requirements.txt
```

### 2. 執行應用程式
```bash
python main.py
```
